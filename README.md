P Address Tracker

The IP Address Tracker lets users find the location, timezone, and ISP details of any IP address or domain. It also shows the user's own IP information on page load and displays the location on an interactive map.

Features

Detect your current IP automatically

Search any IP address or domain

View:

Location

Timezone

ISP

Map marker using Leaflet.js

Fully responsive design for all screen sizes

Smooth UI and hover interactions

How to Run

Clone the repo

git clone <your-repo-link>


Open the folder in your code editor

Run with Live Server (VS Code) or deploy on Netlify/Vercel

IP Geolocation API
